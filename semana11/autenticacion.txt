El login permite identificar a un usuario antes de darle acceso a una aplicación. Gracias a la autenticación, solo las personas autorizadas pueden ingresar a determinadas funciones. Algunas rutas deben estar protegidas porque contienen información privada o permiten realizar cambios importantes. De esta manera se evita el acceso de personas no autorizadas. Además, nunca deben mostrarse públicamente contraseñas, documentos de identidad, teléfonos, direcciones, claves de API ni otros datos sensibles de la comunidad.
