Al ejecutar el comando node server.js, el servidor inicia correctamente y queda esperando solicitudes en el puerto 3000.

En la terminal aparece el mensaje "Servidor ejecutándose en puerto 3000", indicando que el servicio está activo.

Al visitar la ruta /estado en el navegador, el servidor responde con un objeto JSON que informa que el servidor está funcionando y que el servicio corresponde a una API comunitaria.
