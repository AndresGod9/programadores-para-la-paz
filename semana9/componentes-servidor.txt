Sistema operativo:
Es el software principal que administra los recursos del computador y permite ejecutar programas como Node.js. También controla el acceso al hardware y al sistema.

Node.js:
Es un entorno de ejecución que permite utilizar JavaScript en el servidor para crear aplicaciones web y APIs.

Aplicación server.js:
Es el archivo donde se escribe el código del servidor. Allí se definen las rutas y las respuestas que recibirán los usuarios.

Puerto:
Es el número que utiliza el servidor para escuchar las solicitudes de los usuarios. En esta práctica se utiliza el puerto 3000.

Logs:
Son registros que almacenan información sobre el funcionamiento del servidor, los errores y las actividades realizadas. Ayudan a detectar problemas y facilitar el mantenimiento.
